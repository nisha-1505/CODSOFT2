# Sales Prediction

## Objective
To predict product sales based on advertising expenditure using machine learning.

## Dataset
Advertising dataset containing TV, Radio, Newspaper and Sales columns.

## Steps Performed
- Loaded and analyzed the dataset
- Separated input features and target
- Split data into training and testing
- Trained a Linear Regression model
- Evaluated using R2 score

## Algorithm Used
Linear Regression

## Result
The model achieved an R2 score of 0.90, showing strong prediction accuracy.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn
